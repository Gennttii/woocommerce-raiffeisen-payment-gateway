=== WooCommerce Raiffeisen Payment Gateway ===

A WooCommerce Extension that adds the payment gateway "Raiffeisen Payment Gateway"

== Description ==

This extension will add payment gateway "Raiffeisen Payment Gateway" in WooCommerce store.
A certificate provided by Raiffeisen is required in order for this plugin to work (placed at /cert folder).

*This plugin is based on framework by: fadion/Raiffeisen-eCommerce
